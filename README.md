# Drone Delivery Simulation

To use the simulation:
1. Clone the repo
2. Build project
3. Run app executable with input file redirection
```
./app < drone-test-large.txt
```

# Input File Format
```
<n drones>
<m delivery locations>
<m new line separated coordinate pairs>
    eg. 15 -32
    x, y belong to [-100, 100]
```
